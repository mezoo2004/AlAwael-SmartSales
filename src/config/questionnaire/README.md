# Questionnaire Configuration

Edit these files to manage the questionnaire without changing application logic:

- `departments.json`: add, remove, rename, reorder, or update category cards.
- `questions.json`: add, remove, reorder, or update questions and answer options.
- `flow.json`: update injected flow questions and branching rules such as dimensions and glass-related skips.

Question and department order is controlled by the `order` field.
